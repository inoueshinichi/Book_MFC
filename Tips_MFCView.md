### MFC Viewクラス一覧
| View基本クラス | 内容 |
| :-- | :-- |
| CView | ビューの基底クラス. プロジェクト作成時のデフォルトビューはこのクラスの派生 |
| CCtrlView | - |
| CEditView | 簡単なマルチラインテキストエディタを提供するビュー |
| CListView | CListCtrl + ドキュメント/アーキテクチャ |
| CRichEditView | - |
| CTreeView | CTreeCtrl + ドキュメント/アーキテクチャ |
| CScrollView | スクロール機能を自動的に提供するビュー |
| CFormView | ダイアログボックスを持つスクロール可能なビュー |
| CDaoRecordView | - |
| CHtmlView | ウェブブラウザの機能を提供するビュー |
| CHtmlEditView | - |
| CRecordView | - |
| COleDBRecordView | - |
| CPreviewView | - |
| CPreviewViewEx | - |
| CTabView | CMFCTabCtrl + ドキュメント/アーキテクチャ |
| CWinFormsView | - |
